# Combinational-Arithmetic-Circuits
Verilog combinational logic project featuring an 8-bit ripple-carry adder, 4x4 array multiplier, generic K-bit adder, and self-checking testbenches.
## What I Built

- 8-bit ripple-carry adder
- Signed overflow detection
- 4x4 unsigned array multiplier
- Generic parameterized K-bit adder
- Self-checking testbenches

## Tools Used

- Verilog
- ModelSim
- Intel Quartus Prime
- DE10-Lite FPGA Board
